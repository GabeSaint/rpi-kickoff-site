Instructions

Update Package List

To update the package list on Ubuntu, run the following command in the terminal:

sudo apt update

Update Packages

To implement these updates and confirm their installation, run the following command in the terminal:

sudo apt upgrade -y

Install NGINX

To install NGINX, run the following command in the terminal:

sudo apt install nginx -y

Check Service Status

To check if NGINX is running, run the following command in the terminal:

service nginx status

Navigate To NGINX File Directory

To view the files that make up the webpage, run the following command in the terminal:

cd /var/www -y

View Files In The Directory

To view the files that in the directory, run the following command in the terminal:

ls
